Welcome!

This is try-dat, a tutorial that teaches you how to work with datasets using dat. dat is still very much in the early stages of development, but we hope to enable you with some really effective data collaboration, versioning, and syncing workflows.

If you look to your right, you'll see there is a terminal that is connected to a docker container -- that is a computer, in the cloud but accessible right here in your browser!

Below the terminal, there's a filebrowser and an editor. These are connected to the same machine. Every time you type in the editor, try-dat automatically saves the file. Groovy.

When you leave or close this webpage your server will be shut down.
